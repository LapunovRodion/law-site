# Сайт юридического факультета БГУ — law-site

Монорепозиторий нового сайта юридического факультета. Проект разворачивается на своём сервере и использует headless-архитектуру: Strapi как CMS + Next.js фронтенд.

## Цели проекта

- Удобная админка для редакторов (новости, мероприятия, кафедры, сотрудники, документы).
- Быстрый, SEO-дружелюбный публичный сайт.
- Полный контроль над данными и инфраструктурой (самохостинг).
- Чёткая, расширяемая структура контента.

## Технологический стек

- **Frontend:** Next.js (React) + TypeScript + Tailwind CSS (`/frontend`)
- **Backend (CMS):** Strapi 5 (Node.js, TypeScript) (`/backend`)
- **База данных:** PostgreSQL
- **Инфраструктура:** Docker, docker-compose (`/stack`), далее Nginx + HTTPS на продакшене
- **Управление пакетами:** pnpm
- **Node.js:** 22 LTS через nvm (локальная разработка)

## Структура репозитория
law-site/
  frontend/   — Next.js фронтенд (публичный сайт)
  backend/    — Strapi 5 backend (админка + API)
  stack/      — Docker-конфигурация (PostgreSQL и дальше общий стек)
  docs/       — документация (архитектура, модели контента, деплой)
  README.md   — это описание проекта
